      ▸ Common errors & fixes
      

        ❶ "Failed to fetch" — URL must end in /exec not /dev

        ❷ "Script requires auth" — Redeploy: Who has access → Anyone

        ❸ Tab not found — Tab name must match exactly (capitalisation & spaces)

        ❹ Time shows as date — Old script version — create a New deployment

        ❺ After any edit — Always New deployment to get a fresh /exec URL
