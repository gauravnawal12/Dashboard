      ▸ Common errors & fixes
      

        ❶ "Failed to fetch" — URL must end in /exec not /dev

        ❷ "Script requires auth" — Redeploy: Who has access → Anyone

        ❸ Tab not found — Tab name must match exactly (capitalisation & spaces)

        ❹ Time shows as date — You are running an old version of the script.
          Go to script.google.com → Deploy → New deployment (not "Manage deployments").
          Editing an existing deployment does NOT update the running code.

        ❺ After any script edit — Always New deployment to get a fresh /exec URL
